## Projetos em destaque

### Importador de XML NFe/CFe

Sistema para leitura, processamento e importação de arquivos XML fiscais, com foco em performance, checkpoint de importação e persistência em banco de dados.

**Tecnologias utilizadas:**

- C#
- .NET
- SQL Server
- SqlBulkCopy
- XML Reader
- Hangfire

---

### API de Integração e Upload

Projeto voltado para upload, processamento e consulta de arquivos fiscais, utilizando back-end em Python/FastAPI, Docker e arquitetura com workers.

**Tecnologias utilizadas:**

- Python
- FastAPI
- Docker
- Redis
- SQL Server
- Parquet
- Worker assíncrono

---

### Automações e Robôs de Apoio

Scripts para análise, movimentação e comparação de arquivos, consultas em banco e geração de relatórios auxiliares.

**Tecnologias utilizadas:**

- Python
- Pandas
- SQL
- Manipulação de arquivos
- Automação de processos
